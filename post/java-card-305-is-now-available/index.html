자바 카드 3.0.5가 릴리즈 되었습니다

June 5, 2015

새로운 버전의 자바 카드 클래식 에디션이 출시되었습니다. 이제 자바 카드 3.0.5 사양(Specification) 및 개발 키트를 다운로드 할 수 있습니다.

새로운 기능#

이번 릴리즈에는 자바 카드 개발자를 위한 많은 새로운 기능과 Eclipse에서 동작하는 새로운 IDE 플러그인이 포함되어 있습니다.

다음은 새로운 기능을 요약한 것입니다.

  • 뱅킹용 PIN 확장. OwnerPIN 클래스는 오랫동안 EMV 애플리케이션 개발자에게 골칫거리였으며 새로운 PIN 확장은 이러한 문제를 해결합니다. 예를 들어 PIN 개체 할당 후 PIN 표시 시도 횟수를 업데이트 할 수 있습니다.

  • 변수와 배열을 보호합니다. 이 새로운 API를 통해 개발자는 특정 API를 사용하여 외부 공격으로부터 데이터 무결성을 보호할 수 있습니다. 이는 일부 민감한 응용 프로그램의 개발을 크게 단순화할 수 있는 잠재력이 있습니다.

  • 기본 작업을 보호합니다. 이 새로운 API를 통해 개발자는 작업을 두 번 수행하지 않고도 외부 공격으로부터 민감한 작업의 무결성을 보호할 수 있습니다.

  • 정적 암호화를 제공합니다. 이 새로운 API는 단일 작업에 암호화가 필요할 때 암호화 엔진의 사용을 단순화하고 최적화합니다.

  • 타원 곡선 영역 보존(Elliptic curve domain conservation). 이 새로운 기능은 작업 간에 도메인 정의를 보존하여 타원형 도메인 키 세트의 초기화를 단순화합니다.

  • Diffie-Hellman 지원. 키 교환 범위가 향상되었습니다(Improved key agreement coverage).

  • 자바 언어 개선 지원. 자바 카드는 이제 자바7 또는 자바8로 컴파일 할 수 있으며, 자바7에 정의된 언어 개선 사항(예: 바이너리 리터럴 등)이 지원됩니다.

  • 일대다 생체 인식 지원. 생체 인식 API는 생체 인식 샘플을 여러 템플릿과 비교할 수 있는 일대다 생체 인식을 지원하도록 확장되었습니다.

  • APDU 분석을 위한 유틸리티. 응용 프로그램에서 일반적으로 수행하는 간단한 APDU 분석 작업을 수행하기 위해 유틸리티가 추가되었습니다.

이러한 새로운 기능 외에도, 이제 AES-CMAC, PACE, SHA-3, AEAD_CCM 및 AEAD_GCM을 비롯한 새로운 암호화 알고리즘이 지원됩니다.

향후 계획은 무엇인가요?#

새로운 기능 중 일부는 애플리케이션 개발에 상당한 영향을 미칠 수 있으며, 바로 그 내용을 살펴보실 수 있습니다.

  • 자바 카드 3.0.5 사양을 다운로드하여 읽어보십시오.

  • 자바 카드 3.0.5 개발 키트를 다운로드하여, 카드 애플릿의 새로운 기능을 사용해 보십시오.

새로운 기능에 대한 자세한 내용은 본 블로그의 추가 블로그에서 설명할 예정입니다.
